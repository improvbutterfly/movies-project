Welcome to the Movie Recommender!






The Movie Recommendation Project


We built a Natural Language Processing (NLP) movie recommender using Python and the SK-Learn kit. The dataset (cited below) gathered 60,000 movies and 1,000,000 user generated tags from IMBD. An example of the tags for the movie Titanic are:

"drama romance james cameron kate winslet leonardo dicaprio bittersweet catastrophe drama epic historical love true story".

Together, they mentioned actors, directors, descriptions, and more. We cleaned and aggregated the tags grouped by movie_id to create our word soup. Then we made a matrix with the word soup to find the cosine_similarity between the tags. This is the machine learning, NLP aspect, which seeks to compare tags between movies, and return the most similar tag relationships.
